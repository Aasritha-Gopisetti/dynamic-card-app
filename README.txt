# 🃏 Dynamic Card App (Angular)

This is a simple and responsive Angular application for dynamically displaying, editing, adding, and deleting cards. Each card contains a title and description, and users can update the description using a modal popup. The card data is managed using a JSON-like structure, allowing dynamic updates.

## ✨ Features

- ✅ Display cards dynamically from a JSON data structure
- 📝 Edit descriptions using a modal popup
- ➕ Add new cards
- ❌ Delete existing cards
- 🔄 Live UI updates without page reload
- 📱 Responsive design for desktop and mobile

## 🛠️ Technologies Used

- **Angular** (v16+)
- **TypeScript**
- **HTML & CSS**
- **Custom Modal Component**

## ⚙️ Installation & Setup

1. **Clone the Repository**  
   Run: `git clone https://github.com/Aasritha-Gopisetti/dynamic-card-app.git`  
   Navigate into the project folder: `cd dynamic-card-app`

2. **Install Dependencies**  
   Run: `npm install`  
   This will install all required packages listed in `package.json`.

3. **Run the Application Locally**  
   Run: `ng serve`  
   Then open your browser and go to `http://localhost:4200` (if the port is busy, Angular CLI will prompt you to use another).
