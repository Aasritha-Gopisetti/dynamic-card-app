## âš™ï¸ Installation & Setup

1. **Clone the Repository**  
   Run: `git clone https://github.com/Aasritha-Gopisetti/dynamic-card-app.git`  
   Navigate into the project folder: `cd dynamic-card-app`

2. **Install Dependencies**  
   Run: `npm install`  
   This will install all required packages listed in `package.json`.

3. **Run the Application Locally**  
   Run: `ng serve`  
   Then open your browser and go to `http://localhost:4200` (if the port is busy, Angular CLI will prompt you to use another).
